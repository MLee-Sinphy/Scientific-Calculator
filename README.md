# Scientific Calculator

A modern scientific calculator developed as an educational software engineering project.

The objective of this project is not only to provide a clean and functional scientific calculator, but also to serve as a practical environment for studying web development, software architecture, numerical computation, and modern development workflows.

---

## Overview

Scientific Calculator is implemented using standard web technologies and is designed to evolve incrementally. Every new feature is introduced as part of a structured learning process, emphasizing clean architecture, maintainability, and software engineering best practices.

The project intentionally avoids unnecessary dependencies whenever possible, allowing the fundamental concepts behind the implementation to remain clear and transparent.

---

## Current Features

- Basic arithmetic operations
- Parentheses support
- Exponentiation
- Mathematical constants (π and e)
- Trigonometric functions
- Hyperbolic functions
- Logarithmic and exponential functions
- Degree and radian modes
- Keyboard input support
- Responsive interface
- Automatic deployment using GitHub Actions
- GitHub Pages hosting

## Development Workflow

Every modification follows a simple engineering workflow:

```
Edit
    ↓
Commit
    ↓
Push
    ↓
GitHub Actions
    ↓
GitHub Pages
```

Continuous Integration automatically publishes the latest version of the application whenever changes are pushed to the `main` branch.

## Educational Purpose

This repository is part of a broader long-term educational initiative focused on mathematics, scientific computing, software engineering, artificial intelligence, and computational physics.

Rather than being developed as a production-ready calculator from the beginning, the project is intentionally built incrementally so that every architectural decision and implementation detail can be understood from first principles.
